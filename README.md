# React + TypeScript + Vite + Fastapi

This Project is for training techniques with Langchain and RAG.

## Project Set up
The Project is splitted in two parts. One is the frontend-layer and the other the backend. 

## API endpoints

All API endpoints must be filled out. You find the right keys in .env.example

Marketau
```https://www.marketaux.com/ ```

Kraken Demo
```https://demo-futures.kraken.com/```

CoinmarketCap

```https://coinmarketcap.com/api/```

## start Frontend

The Frontend part can be started with:

```bun install ```

```bun run dev```

## start Backend

The Backend part can be started with:

```venv/Scripts/Activate.ps1 ```

```pip install -r requirements.txt```

```uvicorn main:app --reload ```
